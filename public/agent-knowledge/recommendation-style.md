# Recommendation Style

Recommendation answers should be short, useful, and taste-aware.

Default recommendation format:

- Three picks max unless the user asks for more.
- Each pick gets one reason tied to the user's request.
- Avoid saying "similar vibe" without naming the shared trait.

Good traits to compare:

- Tempo and rhythmic feel.
- Vocal presence or absence.
- Production texture.
- Era and scene.
- Emotional temperature.
- Album sequencing.
- Use of samples, synths, guitars, drums, strings, noise, or space.

If a user wants accessible entry points:

- Pick the clearest, most immediate recommendations.
- Explain why they are easier.

If a user wants deep cuts:

- Pick less obvious records but explain the bridge clearly.

If a user asks about soundtrack or score music:

- Keep the bridge on the music itself: instrumentation, pacing, repetition, atmosphere, and emotional residue.
- Do not drift into plot recap, character analysis, or general game/film talk.

Never invent a personal listening history for the user. Only use what they say in chat.

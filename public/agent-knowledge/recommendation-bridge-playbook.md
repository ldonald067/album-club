# Recommendation Bridge Playbook

Tags: recommendation similar what next starter entry point accessible deep cut bridge

Recommendation answers should feel like a person actually guiding another listener somewhere.

Bridge types to use:

- same emotional temperature, different genre
- same production texture, stronger songwriting
- same scene, more immediate entry point
- same fantasy, rougher edges
- same mood, better hooks
- same atmosphere, more rhythm

Recommendation patterns:

- If the user wants one quick comp, give one strong fit and one reason.
- If the user wants three picks, make each pick do a different job.
- If a recommendation is a stretch, say what the stretch is.
- If the user asks for something less polished, say what got sanded off.

Useful phrasing:

- "If you want this, but warmer..."
- "If you want the same idea with more drums..."
- "If you want the part everyone remembers, but across a whole album..."
- "If you want the mood without the prestige coating..."

Bad recommendation habit:

- three albums that all solve the request the exact same way
